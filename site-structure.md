# 社内サイト構成案

モダンなデザインの社内向け情報共有サイトの構成案です。
既存のマークダウンファイルを元に、以下の様なHTMLファイル群でサイトを構築します。

## サイトマップ

- **/index.html** (トップページ)
  - サイトの紹介
  - 各ページへのナビゲーションリンク
    - セットアップガイド
    - 薬歴ガイド
    - このプロジェクトについて

- **/setup.html** (セットアップガイド)
  - `setup.md` の内容を元に生成。
  - **初期セットアップとリモート管理:**
    - PCの命名規則とローカル管理者アカウント
    - ネットワーク設定
  - **トラブルシューティングとメンテナンス:**
    - 一般的な問題と解決策
    - 定期メンテナンス計画
  - **セキュリティとコンプライアンス:**
    - BitLockerによるドライブ暗号化
    - アカウントポリシーの設定
    - 監査ポリシーの有効化
  - **運用・管理のための追加情報:**
    - **便利なコマンド集:**
      - `ipconfig`: IPアドレス設定の確認
      - `ping`: ネットワーク接続の疎通確認
      - `tracert`: 宛先までのネットワーク経路の確認
      - `netstat`: ネットワーク接続状態とポート使用状況の確認
      - `pathping`: ネットワーク経路上のパケット損失の調査
      - `nslookup`: DNS名前解決の確認
    - **セットアップの自動化:**
      - **Batファイル作成入門:**
        - 基本的なコマンド (`@ECHO OFF`, `MKDIR`, `ROBOCOPY`, `START /WAIT`) の使い方
        - 管理者権限で実行するためのチェック方法
      - **サイレントインストール:**
        - MSIインストーラ (`msiexec /i "installer.msi" /qn`)
        - InstallShield (`setup.exe /s /v"/qn"`)
        - Inno Setup (`setup.exe /VERYSILENT /SUPPRESSMSGBOXES`)
        - NSIS (`setup.exe /S`)
    - **便利なショートカットキー:**
      - `Win + X`: パワーユーザーメニュー
      - `Ctrl + Shift + Esc`: タスクマネージャー
      - `Win + E`: エクスプローラー
      - `Win + V`: クリップボード履歴
      - `Win + Shift + S`: 画面切り取り
      - `Ctrl + N`: 新しいウィンドウを開く
      - `Ctrl + T`: 新しいタブを開く
      - `Ctrl + F`: 検索
      - `Win + Arrow Keys`: ウィンドウのスナップ
      - `Win + Shift + Left/Right Arrow`: ウィンドウを別モニターに移動

- **/yakureki.html** (薬歴ガイド)
  - `薬歴.md` の内容を元に生成。
  - レセコンの報酬計算、よくある問い合わせ、クイズなどを掲載。

## デザインと技術要素

- **デザイン:**
  - モダンでクリーンなレイアウトを採用。
  - 十分な余白と読みやすいタイポグラフィを重視。
- **技術:**
  - HTML
  - CSS (場合によってはBootstrapなどのフレームワークを利用)
  - JavaScript (動的な要素が必要な場合)
